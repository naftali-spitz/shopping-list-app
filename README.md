# Shopping List App

Futuristic shopping list web application.

## Stack

- Next.js
- TypeScript
- TailwindCSS
- Framer Motion
- Supabase

## MVP

- Category cards
- Product modal
- Floating shopping list
- DOC export
- Shopping history
- Light and dark mode
- Animated UI

## Goal

Build a polished shopping experience with smooth animations and a simple UX.
